# 블로그 시작(Start my blog)

드디어, 인생 처음으로 블로그를 시작한다.
사실 네이버에서 글을 몇 번 올려본 적이 있긴 한데... 그건 테스트 정도의 수준이었고.
지금부터가 본격적인 시작이라고 말할 수 있을 것이다.

여러 블로그 플랫폼을 고민했었다.

1. 네이버 블로그
2. 티스토리
3. github

각각의 장단점이 있어 고민을 좀 오래 했지만,
결국 내 선택은 github였다.

간단하게 생각해봤을 때.

* 네이버 블로그 -> **네이버**
* 티스토리 -> **카카오**
* github -> **microsoft**

뭐 셋 다 아주 거대한 기업이긴 하지만, 아무래도 카카오나 네이버보다는 마이크로소프트가 압도적으로 크니까...
github블로그를 이용하면 포스팅을 올릴 때마다 commit를 해야 하기 때문에 자연스럽게 잔디밭도 가꿀 수 있게 된다.
commit 및 push 방식으로 포스팅을 해야 하는 점을 불편하게 생각하는 사람들도 많이 있는 거 같은데,
난 딱히 불편하다는 생각은 들지 않았다. 취직할 때 github 주소 하나만 넘겨주면 나에 관한 이력을 전부 볼 수 있게 되는 거니까.
그래서 github를 선택하게 됬다.
___
프로그래밍 관련된 것 외에도 많은 활동 기록을 포스팅해보려고 한다.
* 독서 및 영화 감상문
* 자산 관리
* 평범한 인생 이야기

얼마나 꾸준히 할 수 있을지 모르겠지만, 할 수 있는 한 열심히 해보려고 한다.
그럼 첫번째 포스팅은 여기까지.
